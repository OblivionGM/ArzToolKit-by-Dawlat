# ArzToolKit-by-Dawlat

**Версия**: 1.0
**Автор**: Dawlat  
**Платформа**: [MoonLoader](https://blast.hk/moonloader/)

ArzToolKit — это удобный инструмент для игроков проекта [Arizona RP](https://arizona-rp.com/), автоматизирующий рутинные действия. Скрипт включает набор функций для работы с налогами, охранниками, VIP чатом что делает игровой процесс более комфортным.

## Особенности

- Авто-реклама с настройкой состояния АФК и периода отправки
- Разделение команд VIP-чата
- Автоматическое определение рекламы по ключевым словам
- Пост-обработка рекламного сообщения
- Анти-флуд рекламы с настраиваемым КД
- Авто-оплата обычных и семейных налогов
- Призвать/убрать охранника с помощью одной кнопки
- Быстрый доступ к инвентарю охранника

## Установка

1. Скачайте файл `ArzToolKit v1.0 by Dawlat.lua` из [репозитория](https://github.com/OblivionGM/ArzToolKit-by-Dawlat).
2. Переместите файл `ArzToolKit v1.0 by Dawlat.lua` в папку `moonloader` вашей SAMP-сборки.
3. Перезапустите игру если она была запущена.

## Использование

1. Для открытия для открытия настроек используйте команду `/at`
2. Для оплаты обычных налогов - `/ptax`, семейных - `/pftax`
3. Активация авто-рекламы - `/aa`
